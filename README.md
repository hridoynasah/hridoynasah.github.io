# âš¡ Getting Started (Local Development)

To view or modify this website on your local machine, follow these steps:

## 1. Clone the Repository
Open your terminal and run:

```bash
git clone https://github.com/hridoynasah/hridoynasah.github.io.git
````

## 2. Navigate to the Directory

```bash
cd hridoynasah.github.io
```

## 3. Run the Website

Since this is a static site, you don't need a backend server, but using one is recommended to avoid file-path issues.

### Option A: Python (Recommended)

If you have Python installed:

```bash
# Python 3
python -m http.server 8000
```

Then open your browser and go to [http://localhost:8000](http://localhost:8000).

### Option B: VS Code Live Server

If you use Visual Studio Code, install the "Live Server" extension, right-click `index.html`, and select **Open with Live Server**.

### Option C: Direct File Open

Simply double-click `index.html` in your file explorer.

> Note: some relative links might behave differently depending on browser security settings.
